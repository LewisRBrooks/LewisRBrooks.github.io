Bootcamp 2020: AudioCube

My first programming project, concluding the 2 week Programming Bootcamp at City, University of London. This Processing project demonstrates a comprehensive understanding of the use of variables, loops, 3d graphics, transforming, rotation as well as sound processing. I created a 3D box using nested for loops, and using the Sound library from Processing was able to input songs and analyze them using Fast Fourier Transform (FFT) analyzer and outputting the results to alter the depth of the shapes. User can also interact with the program using the keyboard via [SPACE] for pausing and playing, and Number Keys, [1, 2, 3] to choose which song to play and visualise.

Programming in Java: Spring Coursework Milestone 3

After learning about java for the past 2 semesters at City, University of London, I have been able to create a Game using the city engine and using the various skills that I have been taught over the previous months. With each milestone I have improved the game in some way which then has allowed me to create this final product of Private Tank. It is a 2d platform shooting game, where the aim is to progress and take down the boss. A total of 4 levels, each with varying background and music The players score allows the player to progress to each level and their progress can be saved and loaded via the GUI Also the GUI offers the ability to mute and pause the game, and the loading function allows the player to select a file to load from.
